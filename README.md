# 珀莱雅 · AI 功效护肤导购
> 用科学读懂成分，用数据理解肌肤 —— 让每一次护肤选择都透明而精准

我们选购护肤品时，常常被美好的宣传迷惑了双眼，忽略了**由于地区、人种、遗传导致的皮肤差异**，并且**不同产品的“适应症”不同**。如果能抛弃宣传，**从护肤品成分功效的视角**指导不同肤质的顾客选购商品，那对于消费者而言，将会省时省力省心。

 **在线体验**：https://danbaofan-nancy.github.io/PROYA_AI_consultant/AI_consultant.html



## 项目特色

### 成分科学 · 文献溯源
告别“概念添加”与营销迷雾。我们聚焦护肤品**核心成分的功效机制**，基于公开发表的皮肤医学与成分研究文献，以生物学为基础，将复杂的科学语言转化为可理解的护肤建议。每一种成分的推荐，背后都有科学依据支撑。

### 地域肤质 ·  精准匹配
不同地区的环境气候、不同人种的皮肤特征，对护肤品的响应存在显著差异。我们的网页基于已有数据库，自动抓取不同**地域气候**和**基因相关因素**，归纳总结出导致不同肤质的底层原因，并采用RAG检索增强生成系统，避免大模型产生幻觉，将用户输入的问题与产品成分特性进行多维度匹配，拒绝“一刀切”的标准化推荐。

### 真实评价 · 多维佐证
整合来自**天猫旗舰店**、**小红书**等平台的真实用户评论，通过正负面评价聚合呈现，让“同样肤质的人用了怎么说”成为决策的重要参考。每一份推荐，都有真实用户的实践验证。



## 核心功能

| 功能模块 | 说明 |
|---------|------|
|  **智能产品推荐** | 根据用户输入的皮肤问题，匹配最合适的珀莱雅经典产品 |
|  **成分透明解读** | 解析核心成分的功效机制、适用肤质与科学依据 |
|  **正负面评价聚合** | 汇集天猫、小红书等平台同肤质用户的真实反馈 |
|  **文献依据支撑** | 成分功效引用公开发表的皮肤医学与化妆品学科研文献 |

## 技术实现

- **前端**：原生 HTML + CSS + JavaScript，轻量快速
- **数据层**：基于飞书多维表格构建产品知识库与评价数据池
- **推荐引擎**：规则匹配 + 关键词意图识别
- **部署**：GitHub Pages 持续集成与自动部署
